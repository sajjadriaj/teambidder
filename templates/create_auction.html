JSON Format Requirements

Your JSON file should contain an array of player objects with the following structure:

[
  {
    "name": "Player Name",
    "position": "Position/Role",
    "rating": 85.5,
    "starting_bid": 50000,
    "age": 28,
    "nationality": "Country",
    "experience": "5 years",
    "specialties": ["batting", "bowling"]
  },
  ...
]

Required Fields:

  • name: Player's full name
  • position: Playing position or role
  • rating: Player rating (0-100)
  • starting_bid: Minimum bid amount

Optional Fields:

You can include any additional fields specific to your sport. These will be stored as metadata and displayed in the player profiles.

Examples: age, nationality, experience, specialties, stats, etc.
